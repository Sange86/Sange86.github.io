map 映射 通过键值对来存储数据
可通过set,get来添加获取数据

score.set(“cui”,86);
score.get(“cui”); //86
score.delete(“cui”);//移除
score.has(“cui”);//是否存在

Set 常用做集合=>存储一组不重复的数据
var A = new Set();
var B = new Set([1,2,3]);
var C = new Set([1,2,4,5,5,’5’]);
alert(C); //重复元素在Set中自动被过滤：
可通过add添加元素
C.add(6);
C.delete(4);//删除元素

javascript<一>

substring() 返回指定索引区间的子串
var s = “Hello, world”
s.substring(0,5) // 从索引0开始到5（不包括5），返回’hello’
s.substring(7); // 从索引7开始到结束，返回’world’

indexOf()会搜索指定字符串或者数组中元素出现的位置：
var s = ‘hello, world’;
s.indexOf(‘world’); // 返回7
s.indexOf(‘World’); // 没有找到指定的子串，返回-1
