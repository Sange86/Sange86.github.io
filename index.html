自定义指令directive

代码示例:
var m = angular.module(‘app’,[]);
m.directive(‘mystyle’,[function(){
return {
restrict:”A”,
template:”我爱打水”,
replace:true
}
}])
解析：directive的参数：(1)指令名称
(2)自定义处理函数，返回的是Json数据
属性：
(1)restict:指令的表现形式(A(atrr)表示属性，E(elemt)：元素，C:class)
(2)template:指令在页面的内容

angularJs常见的指令

(1)ng-true-value=”1” 选择时，指定值为1;
(2)ng-false-value=”0” 取消时，指定值为0;
一般配合checkBox 来使用
(3)ng-show:当为true时，显示某元素，反之亦然。
(4)ng-options:下拉选项指令 一般值为：ng-options=’v.value as v.name for v in data’
(5)ng-trim:去掉前后空格(很少用)
(6)ng-if:隐藏元素(删除当前元素，即是从DOM中移除节点，注意与ng-show的区别)

(2)给大盒子设置一个overflow属性
