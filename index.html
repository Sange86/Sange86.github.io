自定义指令directive

代码示例:
var m = angular.module(‘app’,[]);
m.directive(‘mystyle’,[function(){
return {
restrict:”A”,
template:”我爱打水”,
replace:true
}
}])
解析：directive的参数：(1)指令名称
(2)自定义处理函数，返回的是Json数据
属性：
(1)restict:指令的表现形式(A(atrr)表示属性，E(elemt)：元素，C:class)
(2)template:指令在页面的内容

盒子模型

margin塌陷现象
如果一个大盒子中包含一个小盒子，在给小盒子设置margin-top时，大盒子也会一起向下平移。
解决方案：
(1)给大盒子加一个边框
(2)给大盒子设置一个overflow属性
(3)添加浮动

初始bootstrap

在boostrap框架中，大多数的标签的样式都被重新定义了，而不是采用浏览器默认的样式。同时，也封装了很多常用的样式类(class)，在开发时，可直接给标签添加类名即可。以下是常见的类。

排版


(1)text-left: 文本左对齐。 示例：<标签 class=”text-left”>我爱打水</标签>
(2)text-center: 文本居中
(3)text-right: 文本右对齐
(4)list-unstyled: 移除默认样式 (主要对ul li 列表来说)
(5)list-inline: 将元素设置成内联样式(常用于制作导航栏)
