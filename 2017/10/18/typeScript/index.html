初始typeScript

<数据类型>
1 void – 空 一般用于返回值
2 null –null 只能是null
3 undefined –undefined 只有一种值undefined
4 enum(枚举)
*有限的可能性(有限个数)
5 any –变体变量 可赋值任何类型数据

6 类型推测原则(根据初始值推测变量类型)–数组也一样
var a;等价与var a:any
var a=12;等价与var a:number=12;
8 联合类型
例如：var a:number|string
变量a可以是number类型，也可以string类型
<函数>
声明：typeScript是强类型语言，函数参数的个数和类型必须一致
外部变量声明：
declare var $ 告诉编译器 $是在外部什么了，目前无需处理
可选参数
function add(a:number,b:number,c?:number){}
c后面加了? 表示c为可选参数，在调用时，可以不传值。
