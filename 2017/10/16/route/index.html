angular-ui-路由

AngularJS 路由允许我们通过不同的 URL 访问不同的内容。通过 AngularJS 可以实现多视图的单页Web应用（single page web application，SPA）
也就是说可以实现在不刷新页面的情况下，访问不同的模块，类似于选项卡。

(1)定义angular模块，并将ui-route模块注入其中，类似于C++中的继承
var m = angular.module(“app”,[‘ui.router’]);
(2)初始化路由，将$stateProvider服务和$urlRouterProcider服务注入。此部分类似于C++中的构造函数。
m.config([“$stateProvider”,”$urlRouterProvider”,function($stateProvider,$urlRouterProvider){
//默认路由
$urlRouterProvider.otherwise(‘/th’);
//定义路由规则
$stateProvider
//规则1：
.state(‘default’,{
url:’/th’,
template:”页面内容”
//也可以设置模版文件
templateUrl:’xx.html’
})
//规则2：
.state(‘home’,{
url:’/home’,
template:”首页”
//也可以添加控制器，方法跟自定义指令一样
controller:[“$scope”,function($scope){
$scope.name=”我爱打水”;
}]
也可以引用外部的控制器 如controller:’ctrl’
})
}])
