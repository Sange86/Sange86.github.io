初始bootstrap

在boostrap框架中，大多数的标签的样式都被重新定义了，而不是采用浏览器默认的样式。同时，也封装了很多常用的样式类(class)，在开发时，可直接给标签添加类名即可。以下是常见的类。

排版


(1)text-left: 文本左对齐。 示例：<标签 class=”text-left”>我爱打水</标签>
(2)text-center: 文本居中
(3)text-right: 文本右对齐
(4)list-unstyled: 移除默认样式 (主要对ul li 列表来说)
(5)list-inline: 将元素设置成内联样式(常用于制作导航栏)

表格


(1)table: 重定义表格大小或者其他样式
(2)table-striped ：给表格添加隔行着色效果. 需要注意的是：要在有table类的前提下使用时。
(3)table-bordered: 为表格加边框
(4)table-responsive: 设置为响应式表格

按钮


(1)btn: 将元素设置为按钮对象
(2)btn-lg: 大号 改变按钮对象的尺寸大小
(3)btn-sm: 小号
(4)btn-xs: 超小号
(5)btn-block: 将按钮对象设置为块元素，占据一整行
(6)disabled: 禁用

表单和图片


(1)form-control: 给表单元素框添加样式，显示方式为display:block,占一整行
(2)form-group: 给表单元素添加外边距
(3)form-line: 内联表单，让所有表单元素居于一行，作用于form标签
(4)input-group-addon: 表单合组，一般需要配合input-group使用 示例如下：
<标签 class=”input-group”>
<标签 class=”input-group-addon”>￥</标签>
<标签input type=”text” class=”form-control” />
</标签>
(5)form-horizontal:表单水平排列 作用于form标签 一般需要col-sm-xx(1,2..10),control-lable的配合 control-label:表示与父元素样式同步
示例如下：
<!form class=”form-horizontal”>
<!div class=”form-group”>
<!label class = “col-sm-2 control-label”>昵称<!/label>
<!div class=”col-sm-10”>
<!input type =”text” class = “form-control”/>
<!/div>
<!/div>
<!/form>
(6)checkbox & checkbox-inline:占据一行的复选框 和内联复选框
(7)form-control:设置下拉列表

校验状态：has-success(成功状态)，has-error(失败状态)，has-warning(警告状态)

(8)glpyhicon:添加额外的图标 常与glpyhicon-ok(glpyhicon-remove),form-control-feedback ,has-feedback一起使用
(9)img-rounded:将图片设置成圆角矩形
(10)img-circle:圆形处理

(11)img-responsive:响应式处理，当窗口变化时，图片也跟着变化

其他


(1)pull-left:左浮动
(2)pull-right:右浮动
(3)center-block:块级元素居中
(4)clearfix:清理浮动 示例：<-div class=”clearfix”><-/div>,这个div放在需要清理浮动区块的前面即可。
(5)breadcrumb:面包屑组件类，一般用于制作导航分类行 <.ul class=”breadcrumb”><./ul>
(6)pagination:分页栏
(7)pager:上下分页栏
(7)lable lable-default:标签
(8)badge: 徽章，一般运用于信息提醒功能，如未读信息的条数，评论数等.
(9)alert alert-success:警告框组件。示例
<.div class=”alert alert-success”>
这是一个成功警告框。
<.button class = “close” data-dismiss=”alert”>
<.span>&timse<./span>
<./button>
<./div>

按钮组件
(1)dropdown:将元素设置为下拉项。常与dropdown-menu,data-toggle=”dropdown”一起使用。具体示例如下：
<.div clasee=”dropdown”>
<.button>
下拉菜单
<./button>
<.ul>
<.li>首页<./li>
<.li>新闻<./li>
<.li>游戏<./li>
<.li>帮助<./li>
<./ul>
<./div>
(2)btn-group:按钮群组，将多个按钮元素安置在一个盒子内，左右按钮元素具有圆角效果

导航组件


(1)nav:去掉默认的line-style
(2)nav-tabs:标签式
(3)nav-pills：胶囊式
(4)nav-stacked:垂直式 在具有nav-pills的前提下有效

data-


(1)data-toggle:触发类型(元素的切换，如：切换成下拉列表 切换为模态框等
)
(2)data-target:触发节点(元素切换目标)
(3)data-dismiss:关闭元素
(4)data-backdrop:默认值为true,表示背景存在黑灰色透明遮罩，且单击空白背景可关闭弹窗。为false时，则相反。为static时，背景存在黑灰色透明遮罩，且单击空白背景不可关闭弹窗。
(5)data-keyboard:为true时，按esc键，可以关闭弹窗，false则相反
(6)data-show:为true时，初始化时默认显示弹窗，false则相反

(7)data-spy:设置为scroll,将设置容器滚动监听事件
(8)data-offset:默认值为0，固定距离内容滚动容器10像素以内。就高亮显示所对应的菜单
