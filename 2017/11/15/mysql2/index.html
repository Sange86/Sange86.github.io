mysql语句

条件语句查询
select * dbtable where title =”HTML5”;
在where中，数值是不区分大小写的，要想区分，可通过BINARY来区分。

update更新语句
update dbtable set name = “贺娟娟” while id = 3;

删除记录
delete from dbtable where id = num;
当没有where时，执行时，删除整个表数据。

like字句: 查询含有%com字段的数据，字段需要在末尾。
select * from dbtable where author like ‘%com’;

union操作符：作用于合并select语句所得的结果集，并过滤掉重复数据。 如果是union all 则不会过滤重复数据。

排序操作order by…
升序：asc
降序：desc
select * from dbtable where author=”sange” order by id=3 desc;

GROUP BY语句 :将数据按照某种约定进行分组。
select name ,count(*) from dbtable ground by name;
以上是按名字进行分组并统计次数。
WITH ROLLUP 可以实现在分组统计数据基础上再进行相同的统计（SUM,AVG,COUNT…）。
例如我们将以上的数据表按名字进行分组，再统计每个人登录的次数：
SELECT name, SUM(singin) as singin_count FROM employee_tbl GROUP BY name WITH ROLLUP;
注意：coalesce 来设置一个可以取代 NUll(总计) 的名称
SELECT coalesce(name, ‘总数’), SUM(singin) as singin_count FROM employee_tbl GROUP BY name WITH ROLLUP;
